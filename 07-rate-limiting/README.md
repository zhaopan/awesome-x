## 核心原则

- **理解机制** — 了解限流机制比绕过限流更重要
- **预防为主** — 建立规范的操作习惯，避免触发限流
- **及时响应** — 发现限流信号后立即采取措施
- **持续监控** — 定期检查账号状态和指标变化

## 限流类型概览

| 限流类型 | 主要触发条件 | 典型恢复时间 |
|----------|-------------|-------------|
| 发帖限流 | 频率过高、内容违规、敏感词触发 | 1-7天 |
| 互动限流 | 大量关注/点赞/评论、自动化操作 | 24-48小时 |
| API限流 | 调用频率过高、异常请求模式 | 15分钟-24小时 |

## 注意事项

- 限流机制基于账号权重和历史行为动态调整
- 新账号的限制通常比老账号更严格
- 认证账号和非认证账号的限制可能不同
- 平台规则会定期更新，请关注官方最新公告
