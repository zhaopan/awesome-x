## 学习心得

1. **多动手** — 光看不练没用，得实际操作
2. **避坑优先** — 先知道哪些不能做，再学怎么做
3. **看数据** — 别凭感觉，用数据说话
4. **持续学** — 平台规则一直在变，得跟上
