# 账号起号与定位

本目录包含 X 账号起号和定位的完整指南，帮助你从零开始建立一个有清晰定位的账号。

## 内容

- [账号定位指南](01-01-account-positioning.md) — 如何确定账号定位、目标受众和人设风格
- [个人资料优化](01-02-profile-optimization.md) — 头像、简介、背景图等优化建议
- [前30天运营指南](01-03-first-30-days.md) — 从0到1的冷启动指南
- [避坑指南](01-04-pitfalls-to-avoid.md) — 常见运营陷阱和避免方法

## 快速开始

1. 先阅读 [账号定位指南](01-01-account-positioning.md) 确定你的定位
2. 按照 [个人资料优化](01-02-profile-optimization.md) 完善账号设置
3. 按照 [前30天运营指南](01-03-first-30-days.md) 开始运营
4. 阅读 [避坑指南](01-04-pitfalls-to-avoid.md) 避免常见错误

## 核心原则

- **定位先行** — 没有清晰定位的账号很难持续增长
- **人设一致** — 保持人设风格的一致性
- **避坑为先** — 在运营前了解常见陷阱
